\doxysubsection{Detailed Description}
Класс стрелка (\doxylink{class_gunslinger}{Gunslinger}), наследующий от \doxylink{class_character}{Character}. 

Класс стрелка включает в себя методы для получения урона, атаки, использования ультимативной способности, а также уникальные перки, такие как увеличение урона с шансом и восстановление здоровья при атаке.
