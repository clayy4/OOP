\doxysubsection{Detailed Description}
Класс ассасина (Assassin), наследующий от \doxylink{class_character}{Character}. 

Класс ассасина включает в себя методы для получения урона, атаки, использования ультимативной способности, а также уникальные перки, такие как шанс избежать урона и увеличенный урон в зависимости от брони.
