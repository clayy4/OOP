\doxysubsection{Detailed Description}
Класс для генерации случайных чисел с плавающей точкой. 

Использует стандартный генератор псевдослучайных чисел Mersenne Twister ({\ttfamily std\+::mt19937}) и равномерное распределение в заданном диапазоне.
