\doxysubsection{Detailed Description}
Базовый класс \doxylink{class_character}{Character}. 

Класс паладина включает в себя методы для получения урона, атаки и ультимативной способности, а также уникальные перки, такие как благословение, которое улучшает броню и урон, а также шанс на снижение урона.
